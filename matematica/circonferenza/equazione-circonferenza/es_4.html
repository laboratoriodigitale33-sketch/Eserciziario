Esercizio 4 — Circonferenza passante per tre punti con \( c \neq 0 \)

Determinare l’equazione della circonferenza passante per i punti \( A(1,0) \), \( B(5,0) \) e \( C(2,3) \).

Dati

  • \( A(1,0) \)
  • \( B(5,0) \)
  • \( C(2,3) \)

Richiesta

Determinare l’equazione della circonferenza passante per i tre punti dati.

Soluzione

Partiamo dalla forma generale dell’equazione della circonferenza:

\[ x^2 + y^2 + ax + by + c = 0 \]

I coefficienti \( a \), \( b \) e \( c \) sono incogniti. Per determinarli imponiamo il passaggio della circonferenza per i tre punti assegnati.

1. Sostituzione del punto \( A(1,0) \)

\[ 1^2 + 0^2 + a \cdot 1 + b \cdot 0 + c = 0 \] \[ 1 + a + c = 0 \]

2. Sostituzione del punto \( B(5,0) \)

\[ 5^2 + 0^2 + 5a + b \cdot 0 + c = 0 \] \[ 25 + 5a + c = 0 \]

3. Sostituzione del punto \( C(2,3) \)

\[ 2^2 + 3^2 + 2a + 3b + c = 0 \] \[ 4 + 9 + 2a + 3b + c = 0 \] \[ 13 + 2a + 3b + c = 0 \]

4. Sistema dei coefficienti

Otteniamo il sistema:

\[ \begin{cases} 1 + a + c = 0 \\ 25 + 5a + c = 0 \\ 13 + 2a + 3b + c = 0 \end{cases} \]

Sottraiamo la prima equazione dalla seconda:

\[ (25 + 5a + c) - (1 + a + c) = 0 \] \[ 24 + 4a = 0 \] \[ 4a = -24 \] \[ a = -6 \]

Sostituiamo \( a=-6 \) nella prima equazione:

\[ 1 - 6 + c = 0 \] \[ c = 5 \]

Ora sostituiamo \( a=-6 \) e \( c=5 \) nella terza equazione:

\[ 13 + 2(-6) + 3b + 5 = 0 \] \[ 13 - 12 + 3b + 5 = 0 \] \[ 6 + 3b = 0 \] \[ b = -2 \]

5. Equazione della circonferenza

Sostituendo \( a=-6 \), \( b=-2 \), \( c=5 \) nella forma generale:

\[ x^2 + y^2 - 6x - 2y + 5 = 0 \]

6. Forma canonica

Completiamo i quadrati:

\[ x^2 - 6x + y^2 - 2y + 5 = 0 \] \[ x^2 - 6x + y^2 - 2y = -5 \] \[ (x^2 - 6x + 9) + (y^2 - 2y + 1) = -5 + 9 + 1 \] \[ (x-3)^2 + (y-1)^2 = 5 \]

Quindi il centro è \( (3,1) \) e il raggio è \( \sqrt{5} \).

Risultato finale

\[ \boxed{x^2 + y^2 - 6x - 2y + 5 = 0} \]

In forma canonica:

\[ \boxed{(x-3)^2 + (y-1)^2 = 5} \]
