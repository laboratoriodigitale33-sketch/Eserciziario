Esercizio 5 — Circonferenza passante per tre punti con sistema più articolato

Determinare l’equazione della circonferenza passante per i punti \( A(-1,0) \), \( B(3,4) \) e \( C(5,2) \).

Dati

  • \( A(-1,0) \)
  • \( B(3,4) \)
  • \( C(5,2) \)

Richiesta

Determinare l’equazione della circonferenza passante per i tre punti assegnati.

Soluzione

Usiamo ancora la forma generale:

\[ x^2 + y^2 + ax + by + c = 0 \]

Sostituiremo le coordinate dei tre punti per ottenere un sistema lineare nelle incognite \( a \), \( b \) e \( c \).

1. Sostituzione del punto \( A(-1,0) \)

\[ (-1)^2 + 0^2 + a(-1) + b \cdot 0 + c = 0 \] \[ 1 - a + c = 0 \]

2. Sostituzione del punto \( B(3,4) \)

\[ 3^2 + 4^2 + 3a + 4b + c = 0 \] \[ 9 + 16 + 3a + 4b + c = 0 \] \[ 25 + 3a + 4b + c = 0 \]

3. Sostituzione del punto \( C(5,2) \)

\[ 5^2 + 2^2 + 5a + 2b + c = 0 \] \[ 25 + 4 + 5a + 2b + c = 0 \] \[ 29 + 5a + 2b + c = 0 \]

4. Sistema dei coefficienti

\[ \begin{cases} 1 - a + c = 0 \\ 25 + 3a + 4b + c = 0 \\ 29 + 5a + 2b + c = 0 \end{cases} \]

Invece di isolare subito \( c \), conviene sottrarre la prima equazione dalle altre due. In questo modo \( c \) si elimina direttamente.

5. Eliminazione di \( c \)

Sottraiamo la prima equazione dalla seconda:

\[ (25 + 3a + 4b + c) - (1 - a + c) = 0 \] \[ 24 + 4a + 4b = 0 \] \[ 4a + 4b = -24 \] \[ a + b = -6 \]

Sottraiamo la prima equazione dalla terza:

\[ (29 + 5a + 2b + c) - (1 - a + c) = 0 \] \[ 28 + 6a + 2b = 0 \] \[ 6a + 2b = -28 \] \[ 3a + b = -14 \]

Rimane quindi il sistema ridotto:

\[ \begin{cases} a + b = -6 \\ 3a + b = -14 \end{cases} \]

6. Risoluzione del sistema ridotto

Sottraiamo la prima equazione dalla seconda:

\[ (3a+b) - (a+b) = -14 - (-6) \] \[ 2a = -8 \] \[ a = -4 \]

Sostituiamo \( a=-4 \) nella relazione \( a+b=-6 \):

\[ -4 + b = -6 \] \[ b = -2 \]

Ora ricaviamo \( c \) dalla prima equazione:

\[ 1 - a + c = 0 \] \[ 1 - (-4) + c = 0 \] \[ 5 + c = 0 \] \[ c = -5 \]

7. Equazione della circonferenza

Sostituendo \( a=-4 \), \( b=-2 \), \( c=-5 \), otteniamo:

\[ x^2 + y^2 - 4x - 2y - 5 = 0 \]

8. Forma canonica

\[ x^2 - 4x + y^2 - 2y - 5 = 0 \] \[ x^2 - 4x + y^2 - 2y = 5 \] \[ (x^2 - 4x + 4) + (y^2 - 2y + 1) = 5 + 4 + 1 \] \[ (x-2)^2 + (y-1)^2 = 10 \]

Quindi il centro della circonferenza è \( (2,1) \), mentre il raggio è:

\[ r = \sqrt{10} \]

Risultato finale

\[ \boxed{x^2 + y^2 - 4x - 2y - 5 = 0} \]

In forma canonica:

\[ \boxed{(x-2)^2 + (y-1)^2 = 10} \]
